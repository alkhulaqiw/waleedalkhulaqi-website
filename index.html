تعليمات نشر الموقع (Termux + Git) 1. افتح Termux ثم انتقل إلى مجلد المشروع المحلي (أو استنسخه من GitHub إذا لم يكن لديك): cd ~/projects # أو git clone https://github.com/alkhulaqiw/waleedalkhulaqi-website.git cd waleedalkhulaqi-website 2. ضع الملفات (index.html، مجلد ai/، css/، images/...) في جذر المستودع. 3. تأكد من وجود ملف CNAME يحتوي على اسم النطاق: # ملف CNAME waleedalkhulaqi.website 4. أضف، ارتكب، وادفع التغييرات: git add . git commit -m "إضافة القالب الرئيسي ودمج مع RSS وFormspree" git push origin main 5. في إعدادات المستودع على GitHub: Settings → Pages اختر الفرع main و/ (root) ثم احفظ. سيصبح الموقع متاحًا خلال دقائق على: https://waleedalkhulaqi.website 6. ملاحظات مهمة: استخدمت proxy مجاني https://api.allorigins.win لقراءة RSS لتجاوز قيود CORS. يمكنك استبداله بخدمة خاصة أو إعداد Cloudflare Worker لاحقًا. لإرسال نماذج الاتصال بدون خادم استخدم خدمات مثل Formspree أو Zapier أو Make (Integromat). بدّل عنوان endpoint في الكود. لتحسين SEO أضف ملف sitemap.xml وrobots.txt في جذر المشروع.
